# Notes App (MERN)

A simple Notes application with authentication, role-based access (Admin/User), and CRUD operations.

---

## Features
- Register/Login with role selection
- Profile view showing user details
- Admin: Create, Update, Delete notes
- User: View notes only
- Cookie-based session persistence
- React frontend + Express/MongoDB backend

---

## Installation

cd frontend (your folder)
npm install
npm run dev
